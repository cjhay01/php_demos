# php_demos

This repository contains the demos used for the PHP topic in WS101.

# How to run the demos
1. Download the code as zip or clone the repository using `git clone https://github.com/cjhay01/php_demos.git`.
2. Download and install XAMPP from https://www.apachefriends.org/index.html
3. Start the Apache server and MySQL from the XAMPP control panel.
4. Place the demo files in the `htdocs` folder within the XAMPP installation directory. (Usually, this is `C:\xampp\htdocs` on Windows or `/Applications/XAMPP/htdocs` on macOS.)
5. Access the demos through your web browser at `http://localhost/php_demo/php_demo(1/2/3).php`.
